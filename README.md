# My Summer Car music wizard
This app helps My Summer Car/My Winter Car game community manage their custom music playlists with ease.

<img width="800" height="450" alt="image" src="https://github.com/user-attachments/assets/5ed794e7-fd4f-4c29-9002-1051603da4cf" />

## Features
### Transcoder
The app provides the functionality of a **full-fledged transcoder**, which supports all of the common audio formats, which are:
`.mp3`, `.wav`, `.flac`, `.m4a`, `.aac`, `.wma`, `.ogg`
### Wizard
You can conveniently control your playlists in both games: My Winter Car; My Summer Car.

## Tech stack
### Desktop app
- .NET 10, C# 14
- Avalonia UI (Classic.Avalonia theme)
### Transcoder
- Rust 1.93.0
- symphonia 0.5.5
- vorbis_rs 0.5.5
